# Journalism_kotlin
kotlin项目

空暇时间，个人根据聚合上开放的接口整理了一个kotlin简易小项目。

项目包含四个模块：
 1：新闻模块： tab下滑动又包含几个小标题，相亲页面加载的webView
 2：段子模块： 该模块主要接入了聚合上的一些笑话接口，布局简单。
 3：星座模块： 该模块主要可以查个星座的当日运势。当前部分效果值得一阅
 4：我的模块： 该模块主要介绍了作者本人的项目地址
 5：还有侧滑部分：目前还在研发中..
